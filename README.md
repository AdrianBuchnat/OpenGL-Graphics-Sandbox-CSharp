# Custom Voxel Engine 🧊

An ongoing Research & Development (R&D) project built from scratch in **C#**. The goal is not to create a commercial game, but to deeply understand the underlying architecture of game engines, rendering pipelines, and low-level C# mechanics.

### 🧠 My Learning Methodology
This project moves deliberately slowly because I prioritize deep understanding over simply writing code. My workflow consists of:
1. **Reading Official Documentation:** Understanding the theory and math behind a concept (e.g., rendering, matrices, voxel chunking).
2. **Concept Research:** Studying architectural approaches through technical articles and videos.
3. **From-Scratch Implementation:** Writing the code myself to ensure I comprehend every line.
4. **AI-Assisted Concept Verification:** Using AI tools as a "rubber duck" and tutor to clarify complex doubts and discuss architectural trade-offs.

### ⚙️ Current Technical Focus
* **Language:** C#
* **Architecture:** Structuring a scalable, component-based engine architecture from the ground up.
* **Math & Rendering:** Implementing and understanding 3D mathematics, spatial logic, and chunk generation.
